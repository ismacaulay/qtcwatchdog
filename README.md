# qtcwatchdog

```
python qtcwatchdog --proj=helloWorld \
                   --path=C:\proj\helloworld \
                   --watch=C:\proj\helloworld\src \
                   --patterns=*.py;*.cpp;*.h
                   --excludes='tmp'
                   -v
```

## Dependencies
#### Python
- watchdog 0.8.3
